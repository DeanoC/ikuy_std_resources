# Overlord project resource repo for ikuy FPGA dream arcade project

- Zynq 7 ARM/FPGA platforms
- UltraScale+ ARM/FPGA platforms

## Software
### Full bare metal OS 
- Includes advanced Ultrascale+ support including USB3 Host mode
- Full support for display pipes and FPGA modules

### Libraries
- Image loaders
- Compression
- Input
- Sprite systems
